#####    
#####   This sample demonstrates how to record an SVO file with ZED SDK functions.
#####   Optionally, images are displayed using OpenCV.                                
#####
#####   =====================
#####   * Build the program *
#####   =====================
#####    
#####       Open a terminal Recorder directory and execute the following command:
#####           $ mkdir build
#####           $ cd build
#####           $ cmake ..
#####           $ make
#####    
#####   ===================
#####   * Run the program *
#####   ===================
#####    
#####       Open a terminal in build directory and execute the following command:
#####           $ ./ZED\ Recorder 
#####
#####   =====================
#####   * Available options *
#####   =====================
#####    
#####    ________________________________________________________________________________
#####   |                       |                                 |                      |
#####   |        Option         |           Description           |       Argument       |
#####   |=======================|=================================|======================|
#####   | --help                | Display help message            |                      |
#####   |_______________________|_________________________________|______________________|
#####   | -d,--display          | Enable image display            |                      |
#####   |                       | (might slow down the recording) |                      |
#####   |_______________________|_________________________________|______________________|
#####   | -r,--resolution       | ZED Camera resolution           | "0": HD2K            |  
#####   |                       | default 2                       | "1": HD1080          |   
#####   |                       |                                 | "2": HD720           |
#####   |                       |                                 | "3": VGA             |
#####   |_______________________|_________________________________|______________________|
#####   | -f,--filename         | Output filename                 | string               |
#####   |_______________________|_________________________________|______________________|
